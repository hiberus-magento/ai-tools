# Output Templates

This directory contains template examples for the outputs generated by each agent in the RFP Analysis Pipeline.

## Available Templates

| Template | Agent | Description |
|----------|-------|-------------|
| `01_requirements_analysis_template.md` | rfp-analyzer | Structured requirements extraction from RFP |
| `02_estimable_items_template.md` | effort-item-extractor | Atomic work items ready for estimation |
| `03_technical_backlog_template.md` | story-generator | User stories with acceptance criteria |
| `04_detailed_estimation_template.md` | effort-estimator | Hour estimates with human vs AI comparison |

## How to Use

These templates show the expected structure of each agent's output. They are **reference examples**, not required inputs.

The actual format is defined in each agent's definition file (`agents/*.md`) in the "Output Format" section. These templates provide a visual example to help you understand what to expect.

## Customizing Output Format

To customize the output format:

1. Edit the agent definition file (`agents/[agent-name].md`)
2. Modify the "Output Format" section
3. The agent will follow your custom format in future runs

Templates in this directory are for reference only and don't affect agent behavior.
