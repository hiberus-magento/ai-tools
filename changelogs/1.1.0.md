## 🎉 New Features

### PHP Skills (7 new skills)

Complete toolkit for modernizing PHP codebases and improving code quality:

#### 1. **php-dead-code-eliminator**
- Analyzes PHP call graph to identify unreachable code
- Uses Rector for automated removal
- Includes safety checks for magic methods, Reflection API, and dynamic access
- Categorizes findings by confidence level (High/Medium/Low)
- Reduces codebase size and improves maintainability
- **Use when:** Cleaning up legacy code, reducing technical debt, preparing for refactoring

#### 2. **php-generator-memory-optimizer**
- Converts array-building loops to generators with yield
- Reduces memory usage by 99%+ in many cases
- Includes patterns for file processing, database chunking, and pipeline chaining
- Maintains iterator compatibility
- **Use when:** Processing large datasets, streaming data, optimizing memory footprint

#### 3. **php-magic-methods-normalizer**
- Replaces __get, __set, __isset, __unset with explicit methods
- Enables IDE autocomplete and static analysis
- Includes migration strategy with deprecation warnings
- Improves code maintainability and type safety
- **Use when:** Modernizing legacy code, improving IDE support, enabling static analysis

#### 4. **php-database-nplusone-resolver**
- Detects and resolves N+1 query problems
- Implements eager loading, batch fetching, and query optimization
- Supports Doctrine and Eloquent ORM patterns
- Includes detection tools and benchmarking
- **Use when:** Optimizing database queries, resolving performance issues, reducing API latency

#### 5. **php-enum-migration-agent**
- Converts class constants to PHP 8.1+ Enums
- Supports both pure and backed enums
- Includes exhaustive matching with match expressions
- Provides migration path for legacy code
- **Use when:** Upgrading to PHP 8.1+, improving type safety, modernizing constant usage

#### 6. **php-match-expression-converter**
- Converts switch statements to match expressions
- Provides strict comparison and exhaustiveness checking
- Reduces code verbosity significantly
- Includes patterns for return values, assignments, and method calls
- **Use when:** Upgrading to PHP 8.0+, improving code clarity, enabling exhaustiveness checking

#### 7. **php-cryptographic-hash-updater**
- Audits for MD5/SHA1 usage in security contexts
- Replaces with password_hash() using BCRYPT or ARGON2ID
- Includes migration strategy for existing password hashes
- Provides gradual re-hashing on user login
- **Use when:** Security audits, compliance requirements, modernizing authentication

### Magento Skills (4 new skills)

Specialized skills for Adobe Commerce / Magento 2 development:

#### 1. **magento-plugin-interceptor-logic**
- Implements Magento 2 plugins (before/after/around interceptors)
- Configures di.xml with proper sortOrder
- Includes testing patterns and performance considerations
- Provides examples for common interception scenarios
- **Use when:** Extending core functionality, customizing business logic, implementing integrations

#### 2. **magento-graphql-endpoint-resolver**
- Creates schema.graphqls files and PHP resolvers
- Implements ResolverInterface for queries and mutations
- Includes pagination, filtering, and authentication patterns
- Supports headless commerce architectures (PWA, mobile apps)
- **Use when:** Building headless storefronts, creating APIs, exposing custom data

#### 3. **magento-declarative-schema-manager**
- Creates db_schema.xml for table definitions
- Generates whitelist with setup:db-declaration:generate-whitelist
- Replaces legacy InstallSchema/UpgradeSchema scripts
- Provides rollback capability and validation
- **Use when:** Creating database tables, upgrading schemas, modernizing legacy modules

#### 4. **magento-rest-api-service-contract**
- Defines service contract interfaces and implementations
- Exposes via webapi.xml with proper ACL
- Includes complete CRUD examples with authentication
- Follows Magento best practices for API development
- **Use when:** Creating REST APIs, exposing business logic, third-party integrations

### Dockergento Skills (5 new skills)

Streamline development with Hiberus Dockergento environment:

#### 1. **dockergento-database-exporter**
- Extracts and secures database schemas using `hm mysqldump`
- Creates timestamped backups with compression and verification
- Provides restore instructions and rollback procedures
- Implements retention policies
- **Use when:** Before database migrations, before destructive operations, daily backups

#### 2. **dockergento-xdebug-toggle**
- Manages Xdebug activation using `hm debug-on` and `hm debug-off`
- Includes IDE configuration helpers (VS Code, PhpStorm)
- Provides performance impact analysis
- Supports profiling and code coverage
- **Use when:** Debugging PHP code, profiling performance, analyzing code coverage

#### 3. **dockergento-varnish-controller**
- Controls Varnish full-page cache using `hm varnish-on` and `hm varnish-off`
- Tests cache hit ratios and invalidation
- Includes cache warmup scripts and performance comparison
- Supports ESI (Edge Side Includes) testing
- **Use when:** Testing full-page cache, simulating production caching, performance testing

#### 4. **dockergento-shell-executor**
- Executes commands inside containers using `hm bash` and `hm exec`
- Supports Magento CLI, Composer, PHP scripts
- Includes batch operations and error handling
- Provides container environment inspection
- **Use when:** Running Magento commands, executing PHP scripts, managing dependencies

#### 5. **dockergento-mysql-controller**
- Manages MySQL operations using `hm mysql` commands
- Imports SQL dumps, executes queries, manages databases
- Includes performance analysis and optimization
- Provides troubleshooting procedures
- **Use when:** Importing databases, executing queries, database maintenance
